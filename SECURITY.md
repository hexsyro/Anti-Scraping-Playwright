# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to the maintainers. All security vulnerabilities will be promptly addressed.

Please include the following:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact of the vulnerability

## Response Timeline

We aim to acknowledge vulnerability reports within 48 hours and provide a timeline for fixes.
